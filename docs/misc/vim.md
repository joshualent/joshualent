Vim is a fantastic modal text editor known for being fast. It's core, vim motions, have been adapted to many applications, including most Code IDEs. It uses keyboard commands to traverse text quickly using touch typing. Along with boosting productivity, a big selling point of vim is reducing mental overhead when editing text.
